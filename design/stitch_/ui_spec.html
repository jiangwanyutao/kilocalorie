01使用说明

这份 Spec 怎么变成设计稿。

  1. 打开 stitch.withgoogle.com（Google Labs · 免费 · 需 Google 账号）
  2. 先把 §3 全局 System Prompt 复制到 Stitch 的 "Design System / Style Guide" 输入框
  3. 然后逐屏（§7–§17）复制每个 Screen Prompt，生成对应屏
  4. 拿到所有屏后自己拼版为完整原型（Figma / 手动排 PDF 皆可）
Stitch 出图规律
  • Stitch 更懂英文 Prompt，本文档所有 Prompt 均为英文
  • 一次一屏效果最佳；不要试图一段 Prompt 出多屏
  • 结果不满意时改动 Prompt 里的关键词而不是整段重写
  • 把颜色 HEX 写在 Prompt 里比说"暖色"更精准
  • iPhone 15 Pro 尺寸（393 × 852）是最佳默认

  • Ink #EFECE4
  • Accent #E86A44（提亮以适应暗底）

03全局 System Prompt

先把这段丢给 Stitch 建立整体调性。

COPY TO STITCH · SYSTEM STYLE GUIDE
You are designing a mobile PWA app called "千卡日记 (Qianka Diary)", a Chinese-language diet and health diary. All UI is in Simplified Chinese.

DESIGN LANGUAGE:
- Warm, calm, diary-like — NOT dark tech, NOT sterile minimal, NOT sugary cute.
- Feels like a hand-kept notebook printed on ivory paper.
- Confidence in numbers: large tabular numerals feel considered.

COLOR PALETTE:
- Background (light): #FAF9F5 warm ivory paper
- Background alt (cards): #F3F0E7
- Text (ink): #1D1917 warm dark brown-black
- Muted text: #5C5852
- Rule lines: #E3E0D6
- Accent (primary CTA, progress ring, key numbers): #C64B2A soy-orange / persimmon
- Accent soft (badges, chips, tags): #F0E4D5
- Support (positive, protein, on-track): #6C7F3A tea green
- Warn (over-limit): #B3781A honey gold

DARK MODE (must design both):
- Background: #17140F warm dark
- Text: #EFECE4
- Accent: #E86A44 (brighter for dark)

TYPOGRAPHY:
- Chinese font: PingFang SC (system stack)
- Numbers use tabular-nums, semibold, extra weight for hero numbers
- Type scale: 34 hero / 28 title / 22 section / 17 body / 13 caption / 11 label

SHAPES & SPACING:
- Card radius 12–16px, chip 999px pill, button 10px
- Very subtle shadows: 0 1px 3px rgba(29,25,23,0.06). No dramatic float.
- Base grid: 8pt

ICONS:
- Line style, 1.5px stroke, 24x24, single color from ink or muted.
- Phosphor / Lucide regular variant. No emoji, no multicolor icons.

TARGET DEVICE:
- iPhone 15 Pro (393 × 852), iOS 17 Safari as PWA.
- Design bottom safe area for home indicator (34pt).

APP STRUCTURE:
- 5-tab bottom bar: Home 首页 / Record 记录 / [+] center capture / AI 搭子 / Me 我的
- Center [+] is a raised circular button in accent color.

MICROCOPY VOICE:
- Chinese, warm-friendly-quiet, not preachy.
- Never say "打卡" as heavy gamification; prefer "记一笔" "存进日记".
- Numbers speak. Copy stays out of the way.
💡 用户第一次进 Stitch 时先输入这段，后续每个单屏 Prompt 都会继承这个 style guide

18交互规范

动画 · 触觉 · 手势 · 状态。

动画时长与曲线

  • 快速反馈：150ms · cubic-bezier(0.4, 0, 0.2, 1)
  • 页面切换：300ms · iOS 系统曲线
  • 数字滚动：600ms ease-out（卡路里从 0 滚到 480）
  • 环形加载：1.2s 无限（描边 dash 变化，不旋转）
  • respect prefers-reduced-motion：降级为 crossfade

触觉反馈（iOS）

  • 轻按选择：UISelectionFeedback
  • 成功记录：UINotificationFeedback(.success)
  • 删除/拒绝：UINotificationFeedback(.warning)
  • 拍照成功：UIImpactFeedback(.medium)
  • PWA 用 navigator.vibrate([10])（iOS 部分支持）

手势

  • 左划食物条目 → 显示"删除"
  • 下拉刷新首页
  • 底部 sheet 支持下滑关闭
  • 头像长按 → 快速切换人设

状态命名（每屏必备）

  • 初始 · 加载中 · 无数据（EmptyState）· 数据展示 · 错误 · 无网络
  • 断网时顶部条："无网络 · 已切离线，记录会稍后上传"

iOS PWA 特殊考虑

  • 安全区：env(safe-area-inset-top/bottom)
  • 加桌面后无地址栏 · 自定义 splash 用 apple-touch-startup-image
  • 横屏禁用（orientation: portrait）
  • Pull-to-refresh 需自实现（默认 Safari 会触发返回上一页）
